# crosslead-adapters 
Common data adapter functionality across web server and background workers


## Install

```bash
$ npm install --save crosslead-adapters
```


## Usage

```javascript
var crossleadAdapters = require('crosslead-adapters');
crossleadAdapters(); // "awesome"
```
